# Task: Blackjack Card Animations

## User Request
Add smooth card animations (slide-in + 3D flip) to the Stack.IT blackjack game in both solo and multiplayer screens. Cards slide in face-down, flip to reveal, and hand values update only after each flip completes.

## Codebase Summary
- **Stack**: Flutter 3.11+ / Dart, deployed as web app via Cloudflare Pages
- **Structure**: `frontend/lib/blackjack/` contains solo screen, multiplayer screen, lobby, and WebSocket client. No existing `widgets/` subdirectory.
- **Conventions**: Single-file screens as `StatefulWidget`, inline `_build*` methods, `Map<String, dynamic>` for card data (`{rank, suit}`), theming via `AppTheme` passed as prop. No animation framework or controllers currently used in blackjack code.
- **Critical Findings**: None. Flutter SDK ^3.11.0 supports all required animation APIs (`AnimationController`, `Matrix4` transforms). No external animation packages needed.

## Relevant Files
| File | Purpose |
